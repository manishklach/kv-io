# Kernel Patches

This directory contains local RFC/POC patch artifacts for KV-IO.

The current focus is an initial local patch stack:

- `0001-rfc-kvio-mq-deadline-decode-priority.patch`
- `0002-rfc-kvio-block-request-classification.patch`
- `0003-rfc-kvio-debugfs-scheduler-stats.patch`

These patches are intended for local validation and benchmark-driven iteration.
